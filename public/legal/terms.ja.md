# **Flaxia SNS 利用規約**

**施行日：** 2026年 3月12日

## **第1条 同意**

アカウントを作成することで、本規約に同意したものとみなします。同意いただけない場合はFlaxiaをご利用いただけません。

## **第2条 利用資格**

Flaxiaを利用するには13歳以上である必要があります。登録することでこの条件を満たすことを確認したものとみなします。

## **第3条 アカウント**

- アカウントの一切の活動についてユーザー本人が責任を負います
- ユーザー名は登録後に変更できません
- 認証情報を他者と共有しないでください

## **第4条 投稿コンテンツ**

- 投稿したコンテンツの所有権はユーザーに帰属します
- 投稿することにより、Flaxiaに対してコンテンツを保存・表示するための非独占的・無償のライセンスを付与したものとみなします
- 投稿はデフォルトで公開されます

## **第5条 禁止事項**

以下の行為を厳禁とします。

| カテゴリ | 具体例 |
| ----- | ----- |
| スパム、迷惑広告 | 自動投稿、宣伝、妨害目的の大量投稿 |
| ヘイトスピーチ | 人種・性別・宗教・国籍・性的指向などを理由とした差別的、あるいは攻撃を示唆するコンテンツ |
| 著作権侵害 | 権利を持たない実行可能ファイル・メディアのアップロード |
| マルウェア、悪意ある実行可能コンテンツ | ブラウザの脆弱性の悪用・ユーザーへの危害・詐欺を目的とした`.swf`・`.js`・`.wasm`等のファイル |
| プライバシー侵害 | 本人の同意なく他者の個人情報を投稿する行為 |
| AIトレーニング | 本サービス内のコンテンツを、機械学習、深層学習、人工知能の開発・学習、または生成AIの入力データとして利用（解析、複製、抽出、加工等を含む）する行為 |

以上の行為を発見した場合、速やかに通報することができます。
ご協力をお願いいたします。

## **第6条 NSFWコンテンツ**

- NSFWコンテンツの投稿は禁止しませんが、`#nsfw`・`#r18`タグの付与を必須とします
- 適切なタグなしにNSFWコンテンツを投稿することは本規約違反となります。
- NSFWタグ付きコンテンツはデフォルトで非表示となります。ログイン中のユーザーはSettingsからオプトインできます

## **第7条 対応方針**

- 運営者は本規約に違反するコンテンツの削除およびアカウントの停止を行う権利を有します
- 現在の段階では正式な異議申し立てプロセスはありません。運営者に直接ご連絡ください

## **第8条 実行可能コンテンツの免責**

- Flaxia上で実行可能なファイル（`.swf`・`.js`・`.wasm`等）はサンドボックス内でクライアント側にて実行されます
- サードパーティの実行可能ファイルの互換性・安全性について運営者は一切の保証をしません
- 実行可能コンテンツの実行はユーザー自身の責任において行ってください

## **第9条 サービスの提供**

- Flaxiaは現状有姿で提供され、稼働率の保証はありません
- 運営者はいつでもサービスを変更・終了する権利を有します
